## BlueProject

This project was made with the intention of having a web application that is interactive, it's a side project design to improve skills in Angular and Symfony.

'marco' is the character you are playing, he is living in Lattes and currently he is quite hungry, so let's order some food.

All the assets come from [pngegg.com](https://www.pngegg.com/)  
The production version of this project stand [here](https://damienbousquet.dev/)

Regarding the database, it use a multi-tenant design architecture working with session (made with php). When a user enter the application or refresh it, it will create a new session.

This is the public repository, i have a private one on gitlab (hence the low number of commits).
